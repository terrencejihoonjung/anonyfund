## About The Project

This project was my submission for Major League Hacking's "Web3Apps" hackathon that took place late December. I decided to use the Flow Client Library to create a decentralized, crowdfunding platform where users can be anonymous and participate in fundraisers using cryptocurrency ($FLOW tokens). After connecting their wallets, users are authorized via their wallet address and are able to create campaigns/donate to campaigns. When creating campaigns, users can enter detailed information and upload images for their campaign posts. 

<br/>

### Built With

* Typescript + React.js
* TailwindCSS
* Node.js + Express.js
* Firebase Realtime Database/Storage
* Flow Client Library
  
<br/>

## Getting Started

Fork the project, download dependencies and run <strong>npm run dev</strong> while in both the client and server directories.

### Prerequisites

Download Dependencies in both the client and server directories.

```sh
npm install
```
